# Sidan Smart Reporting System

## الوصف
نظام سِدان هو نظام ذكي لاستقبال البلاغات الصوتية في بيئات الازدحام مثل موسم الحج.  
يتيح للمستخدم تسجيل بلاغ صوتي بمختلف اللغات، ثم يقوم النظام بتحويله إلى نص، ترجمته إلى العربية، وتحليله لتحديد نوع الحالة ومستوى الخطورة واقتراح إجراء مناسب.

## المميزات
- تسجيل بلاغ صوتي
- دعم لغات متعددة بشكل مبدئي
- ترجمة البلاغ إلى العربية
- تحليل نوع الحالة (صحية، ازدحام، ضياع، حريق)
- تحديد مستوى الخطورة
- اقتراح إجراء ميداني
- حفظ البلاغات في سجل

## التقنيات المستخدمة
- HTML
- CSS
- Tailwind CSS
- JavaScript
- Web Speech API
- Speech Synthesis API
- LocalStorage
